# nagi — デザイン方針

## 方向性

**Photoreal Nature × Calm Interaction.**

抽象的・幾何学的・宇宙的な演出ではなく、**実在する自然を写実的に描く**ことを第一に置く。根拠はバイオフィリア（人間は本能的に自然の映像・音に癒される）で、長時間瞑想セッションに耐える映像は刺激的でない自然しかない。

## ビジュアル基準

### やる

- 写実 / 半写実の自然（焚き火、波、森、雨、星空）
- 暖色 × 深い闇、または冷色 × 薄明（シーンに応じて）
- ゆらぎ・呼吸・揺れの**低周波モーション**（0.5〜2Hz 程度）
- 映像と同期したハプティクス・音の**三位一体**

### やらない

- 高コントラストの点滅 / 高速モーション
- RGB 分離 / グリッチ系エフェクト
- ネオン / サイバー / 宇宙トンネル系（抽象演出）
- 派手なパーティクルバースト（タッチ反応の「火の粉」程度に留める）

## 参考（Visual Target）

- **Blender 焚き火チュートリアル**（カサハラ CG）— https://note.com/real_ruff9813/n/n814554e153af
  - **ビジュアルの到達点**として参照。ただし動画素材としては使わない（procedural Metal で再現する方針）。
- **Apple tvOS Aerials** — Apple 自身がネイティブで達成しているアンビエント映像の上限
- **スタジオジブリの焚き火作画**（紅の豚・もののけ姫）— "ゆらぎのリズム" の美学
- **Shadertoy の fire/flame 系シェーダー** — GLSL の定石（FBM 層化、色温度勾配）を Metal に移植
- **Calm / Oak / Portal** — マルチセンサリー瞑想アプリの先行例。触覚統合の深さで超える。

## 参考にしない（理由付き）

プロジェクト進行中に候補に上がったが**本線に採用しない**もの:

| 候補 | ジャンル | 不採用理由 |
|---|---|---|
| Living Nebula (Three.js) | fbm ベース星雲 | 抽象宇宙、覚醒刺激側 |
| Warp Drive (Three.js) | RGB 分離トンネル | 高コントラスト、瞑想に不向き |
| ATC shader (WebGL2) | raymarch ループ | 認知負荷が高い |
| Abstract Glassy (WebGL2) | SDF メタボール | 冷たい印象、自然感なし |

**ただし Onboarding / Paywall での演出的使用**は保留。主役シーンには使わない。

## カラー / ムード（暫定）

| シーン | ベース | アクセント | ムード |
|---|---|---|---|
| 焚き火 (Relax) | 深い黒 + オレンジ | 火の粉の白・黄 | 暖かい、孤独、安堵 |
| 波 (Sleep) | 深い青 + 紺 | 月光の白 | 涼しい、反復、眠気 |
| 森 / 夜空 (Breath) | 藍・深緑 | 星 / 蛍の点光 | 静寂、呼吸、広がり |

正式なカラートークンはシーン素材が揃ってから定義する（素材のトーンに合わせる方が自然）。

## タイポグラフィ

- 画面 UI は**情報最小**を原則。シーン再生中はテキストを極力消す。
- 呼吸ガイド等で必要な文字は**半透明のサンセリフ**。フォントは暫定で SF Pro、決め打ちはシーン確定後。

## インタラクション設計

### 入力

- **タップ**: 単発の反応（焚き火→火の粉、波→一滴の波紋）
- **長押し**: 持続反応（炎が大きくなる、波が広がり続ける）
- **ドラッグ**: 位置追従反応（指先の周囲が反応）

**使わない**: ピンチ / スワイプ（瞑想中は指を動かしすぎない）。

### フィードバック

- **視覚** ≤ 50ms（Metal uniform 経由の即時反映）
- **触覚** ≤ 30ms（Core Haptics 事前ロード済みパターン）
- **音** ≤ 50ms（PHASE 事前ロード済みサンプル、空間定位）

この 3 つが揃って初めて「触れた」感覚になる。片方でも欠けると違和感。

## 音のデザイン

- 環境音は**ループ可能な実録音**を基本にする（Blender 生成のベース映像に実録音を重ねる）
- タッチで発火する音は**短いサンプル**（パチッ、ピチャ、サラッ）を PHASE で空間配置
- BGM / メロディは**入れない**（マインドフル系は音楽で覚醒すると本末転倒）

## 触覚のデザイン

- パターンは**シーン毎に 2〜3 種**まで絞る（多すぎると耳障りになる）
- 強度は**ピアニッシモ〜メゾピアノ相当**（強い振動は瞑想の邪魔）
- 映像・音と**ズレたら不快**なので、同期は厳守
