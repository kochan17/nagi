# nagi

> 凪のように衝動を鎮める — 衝動抑制ASMRモバイルアプリ

## Concept

SNS（YouTube/Instagram/TikTok）を見たい衝動や暴飲暴食の衝動が来たとき、触覚的・聴覚的な癒しで衝動を逃がすモバイルアプリ。

## Why

- 現代人はSNSや食の衝動に常にさらされている
- 「我慢する」のではなく「代替の快」で衝動を鎮める方がサステナブル
- ベンチマーク: Feelsy（月$50K、DL約4万/月）が証明した「Sell the feeling, not the feature」の市場

## Vision

嵐のような衝動が来たとき、nagiを開く。ASMRの音と触覚フィードバックで心が凪いでいく。機能ではなく感覚を売る。衝動の波が引いた後に残る、静かな満足感を届ける。
